# Fuckify

> Fuckify your bash commands to the moon and beyond

# Usage

```sh
fuckify ...command
```

Or

```sh
fuckify 'command"
```

# Dependencies

- Bash
- Coreutils
- XZ compressor/decompressor
- Base64 en/decoder
- OpenSSL
